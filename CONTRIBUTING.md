# 🤝 Panduan Kontribusi

Kami senang Anda tertarik untuk berkontribusi pada pengembangan **Oemah Bu Liek Management System**!

## 🚀 Cara Berkontribusi

1. **Cari atau Buat Isu**: Sebelum memulai, pastikan perubahan yang Anda usulkan sudah ada di daftar [Issues](../../issues) atau buat baru untuk diskusi.
2. **Fork Repositori**: Lakukan fork ke akun GitHub Anda.
3. **Pengerjaan di Branch Baru**:
   - `git checkout -b feature/nama-fitur` atau
   - `git checkout -b bugfix/nama-bug`
4. **Commit dengan Pesan Jelas**: Gunakan format yang mudah dipahami (misal: `feat: Menambahkan filter laporan berdasarkan tanggal`).
5. **Kirimkan Pull Request**: Sertakan penjelasan tentang apa yang diubah dan mengapa.

## 🛠 Aturan Pengembangan
- Pastikan kode mengikuti standar Laravel (PSR-12).
- Gunakan Tailwind atau Bootstrap sesuai yang sudah diimplementasikan di project.
- Jangan mengunggah file sensitif (seperti `.env`) ke repositori publik.

Terima kasih atas dukungannya!
